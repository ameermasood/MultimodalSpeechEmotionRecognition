# Results

This folder is reserved for selected experiment results, summary tables, and
figures that are useful for understanding the project.

Generated outputs from full training or evaluation runs should stay out of git
unless they are intentionally curated for the portfolio.

Suggested structure:

```text
results/
+-- zero_shot_esd/
+-- zero_shot_iemocap/
+-- esd_adapters/
+-- iemocap_adapters/
+-- summary/
```

Large prediction files, raw logs, and checkpoint artifacts should remain local.
